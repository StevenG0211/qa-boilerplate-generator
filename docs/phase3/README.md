# Phase 3 — Presets

Phase 3 is dedicated to presets and JSON import.

## Scope

- official preset schema
- official preset JSON files
- client-side JSON upload and validation
- preset application state
- preset selector UI
- CI validation for preset files

## Out of Scope

- BDD presets
- raw dependency uploads
- VS Code extension work
- accounts or saved cloud projects
